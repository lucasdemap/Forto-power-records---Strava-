For run this application flow these following steps

STEP 1. Install Python on you system. Goto https://www.python.org/ 

    and download it as you os requirement.

STEP 2. Once python is Install please install the require packages.

        Go to the folder where the app.py exists.

        Then run the command

        $pip install -r requirements.txt

STEP 3. If you can run the applciation using command 

        $python app.py
        
STEP 4. Now server start execution and it will server at you localhost


        url :  127.0.0.1:5000

    Go to the Above url you can see the application running on your localhost.
